MANUAL DO ESCRITURÁRIO ESCOLAR
==============================

COMO USAR:
  Abra o arquivo index.html com o Google Chrome, Edge ou Firefox.

COMO EDITAR CONTEÚDO:
  1. Edite o arquivo HTML do módulo desejado em /modulos/
  2. Atualize o campo updatedAt e version no meta JSON interno
  3. Atualize /data/indice.json com as novas informações
  4. Atualize /data/versoes.json com o changelog da alteração
  5. Reabra index.html no navegador

DEPENDÊNCIAS (arquivos vendor — baixar manualmente):
  Veja /assets/vendor/adminlte/INSTALAR.txt
  Veja /assets/vendor/fusejs/INSTALAR.txt
